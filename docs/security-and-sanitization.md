# Security and Sanitization

Before publishing a local setup:

- Search for `ghp_`, `sk-`, `OPENAI_API_KEY`, `GITHUB_TOKEN`, `Authorization`.
- Replace absolute user paths with placeholders.
- Do not commit plugin cache contents.
- Do not commit local browser profile paths.
- Do not commit private skills, documents, papers, or datasets.

Use example registries in the public repository and keep private overlays local.
